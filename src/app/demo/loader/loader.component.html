loading http拦截UI展示

该处并未涉及到 http 的内容，是纯UI功能的实现。本功能有 service、ngx-bootstrap中的modal、css、html一起完成，其借助于 ngx-bootstrap 的 modal 弹框完成

http拦截器：该http拦截器实现了展示文字，该文字可以自己定义，通过HttpHeaders传到拦截器上，拦截器接收文字后移除文字并克隆当前请求，这样就实现了传送文字在loading UI中展示，并且不会影响http请求

解决方案：该http拦截由 service、组件、ngx-bootstrap/modal 一起完成

http 拦截结果展示

http 拦截结果展示是通过 toastr 实现，在 angular 中使用 toastr 时，需要完成以下步骤：

  1. npm install ngx-toastr --save
  2. npm install @angular/animations --save
  3. 在 angular.json 文件下的 styles 属性中引入 "./node_modules/ngx-toastr/toastr.css"
  4. 在根模块中引入 BrowserAnimationsModule 和 ToastrModule，其分别来源于 '@angular/platform-browser/animations' 和 'ngx-toastr'，其中 ToastrModule 需要 forRoot() 一下
  5. 在需要提示信息的地方引入 ToastrService（来源于 'ngx-toastr'），使用该服务的方法展示提示信息
  6. 此时，看不到提示信息，因为此时的提示信息容器的透明度为100%，需要修改一下其透明度，在 style.css 中修改提示容器的样式
